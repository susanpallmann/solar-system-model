Solar System Model

This is a model solar system based on real data from NASA. Although much of this model is designed to be proportional, some liberties had to be taken in order for all objects to be visible! As such, while the planetary distances from the sun are proportional to each other, they are not proportional to the sizes of the planets (also proportional to each other). Additionally, the sun is made smaller as initially it took up the entire page above the fold!
